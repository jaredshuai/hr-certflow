# Triage 标签

技能用五个标准的 triage 角色来描述状态。这个文件把这些角色映射到本仓库 issue 追踪器里实际使用的标签字符串。

| 技能里的标签 | 本仓库使用的标签 | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 维护者需要评估这个 issue |
| `needs-info` | `needs-info` | 等待提报者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 已充分明确，可交给离线 agent 处理 |
| `ready-for-human` | `ready-for-human` | 需要人工实现 |
| `wontfix` | `wontfix` | 不予处理 |

当技能提到某个角色（例如“打上可离线 agent 处理的 triage 标签”）时，用这张表里对应的标签字符串。

若实际使用的词汇不同，改右列即可。
